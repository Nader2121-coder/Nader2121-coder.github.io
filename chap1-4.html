Où écrire le style CSS > Style externe



Pour relier une feuille de styles externe à un fichier HTML,
vous utiliserez la balise <Link> dans l’entête de ce fichier
(ici, la feuille de styles a pour nom design.css) :

<link rel="stylesheet" href="design.css" />

• La balise <Link> avertit le navigateur qu’il doit chercher
un document situé à l’extérieur de la page HTML.
• L’attribut rel="stylesheet"
 précise que le document en
question est une feuille de style externe.
• L'attribut href indique l’emplacement du fichier CSS à
utiliser.
